ika
===

Service bot daemon for http://ozinger.org/

Work in progress.


Dependency
----------

* Python >= 3.4.0
* see ``requirements.txt``


How to run
----------

.. code-block:: console

   $ pip install -r requirements.txt
   $ cp config.json.example config.json
   $ vim config.json # Or any of your favourite editor except emacs
   $ python run.py
